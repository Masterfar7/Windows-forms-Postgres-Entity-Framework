## Описание
Приложение предоставляет графический интерфейс для выполнения CRUD-операций (создание, чтение, обновление, удаление) над таблицей сотрудников в базе данных PostgreSQL.
## Стек технологий
- C#
- Windows Forms
- PostgreSQL
- Entity Framework Core
## Функциональность
- Просмотр списка всех сотрудников в таблице
- Добавление нового сотрудника (имя, должность, зарплата)
- Обновление данных существующего сотрудника по ID
- Удаление сотрудника по ID
- Поиск сотрудника по ID
- Обновление отображения таблицы
Все операции с базой данных выполняются асинхронно.
